Электронная библиотека представляет собой базу данных всех электронных ресурсов университета. С помощью данного ресурса можно осуществлять поиск необходимой пользователю литературы по названию или автору, посмотреть рекомендуемые преподавателем источники и посетить выставку библиотекарей.


Описание пользователей системы:


Студент — пользователь системы с наименьшими правами доступа. Студент имеет возможность производить поиск по имеющимся ресурсам. О каждом студенте хранится следующая информация:

  • имя,
  • номер зачетной книжки,
  • группа,
  • направление подготовки.

    • Преподаватель — пользователь системы, у которого есть возможность добавлять новые ресурсы, просматривать активность студентов в данной библиотеке, которым читает соответствующие дисциплины. Также преподаватель имеет возможность рекомендовать студентам некоторую литературу. О каждом преподавателе хранится следующая информация:

  • имя,
  • стаж.

    • Библиотекарь — пользователь системы с возможностями выше, чем у преподавателя. Пользователи данной группы имеют возможность добавлять новые ресурсы, просматривать активность каждого пользователя и создавать выставки для студента и преподавателя. О каждом библиотекаре хранятся следующие сведения:

  • имя,
  • уникальный идентификатор.

    • Ресурс — сущность системы, которая является книгой, методическими указаниями, периодическим изданием или статьёй. Каждый ресурс характеризуется своими признаками:

  • Идентификатор
  • Название
  • Список авторов

    • Выставка — объект системы, который содержит в себе определенное количество ресурсов, объединённой одной тематикой. О каждой выставке хранится следующая информация:

  • Идентификатор
  • Название
  • Дата начала
  • Продолжительность.

    • Каждая имеющаяся книга, характеризуется следующими параметрами:

  • авторы,
  • название,
  • дата издания,
  • количество страниц,
  • издательство.
